Campus Resource Reservation System

Developing a secure full-stack platform for campus resource reservations. The system allows authenticated users to book shared facilities such as labs, equipment, and auditoriums. Google OAuth2 integration ensures that only @charusat.edu.in users can access the platform. The application supports role-based access (Admin, Faculty, Student), a structured booking approval process, JWT authentication, and strong session security.
